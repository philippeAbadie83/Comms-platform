# Comms Platform

Base operativa para el envío automatizado de correos y futuras campañas de comunicación en Hidrobart.

Este repositorio documenta el motor de comunicaciones implementado con Power Automate, Excel Online y Outlook, junto con plantillas HTML y el modelo de datos que permite su operación.

El objetivo es habilitar una capacidad reutilizable de comunicaciones (aniversarios, avisos, marketing, newsletter y campañas) administrable por negocio y escalable en el tiempo.

Estado actual: motor de envío base operativo.

Este repositorio actúa como fuente de verdad (source of truth) de:
- arquitectura del motor
- modelo de datos
- plantillas de correo
- lineamientos de operación
- evolución de campañas

La documentación técnica se encuentra en `docs/`.
