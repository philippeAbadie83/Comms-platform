# Comms Platform

Base operativa para el envío automatizado de correos y futuras campañas de comunicación en Hidrobart u otros.

Este repositorio documenta el motor de comunicaciones implementado con Power Automate, Excel Online y Outlook, junto con las plantillas y lineamientos necesarios para su evolución.

El objetivo es habilitar una capacidad reutilizable de comunicaciones (aniversarios, avisos, marketing, newsletter, campañas) administrable por negocio y escalable en el tiempo.

Estado actual: motor de envío base operativo.

La documentación técnica y modelos de evolución se encuentran en la carpeta `docs/`.
